Calculo Difal - Base Unica (DF)

Quando utilizar o cálculo do DIFAL por Base Única:
Utilize esta modalidade quando a operação for interestadual e o consumidor final for não contribuinte de ICMS, ou para o estado do DF. Este cálculo é mais simples e direto, aplicando a diferença entre as alíquotas interna do destino e interestadual sobre o valor total da nota fiscal.

Calculo DIFAL - Base Dupla (AC, AP, BA, CE, ES, MA, MG, MS, MT, PA, PI, PR, RJ, RN, RO, RR, RS, SP, TO)

Quando utilizar:
Use o cálculo por Base Dupla quando o estado de destino exigir o ICMS "por dentro" (AC, AP, BA, CE, ES, MA, MG, MS, MT, PA, PI, PR, RJ, RN, RO, RR, RS, SP, TO).

Cálculo DIFAL - Base Dupla (AL, GO, SC)

Quando utilizar:
Use este cálculo quando o estado de destino for AL, GO ou SC, que utilizam a fórmula baseada na diferença entre alíquotas "por dentro".

Cálculo DIFAL - Base Dupla (SE)

Quando utilizar:
Use este cálculo quando o estado de destino for **Sergipe (SE)** , que utiliza fórmula própria baseada na diferença entre alíquotas, com divisor especial.

Cálculo DIFAL - Base Dupla (AM, PB, PE)

Quando utilizar:
Use este cálculo quando o estado de destino for AM, PB, ou PE, que utilizam uma fórmula ajustada com ICMS de origem subtraído antes da base por dentro.
